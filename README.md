# Homebrew Tap

Homebrew tap for productdevbook apps.

## Installation

```bash
brew tap productdevbook/tap
```

## Available Apps

### PortKiller

Menu bar app to find and kill processes running on open ports.

```bash
brew install --cask portkiller
```

**Requirements:** macOS 15.0+ (Sequoia), Apple Silicon

[GitHub Repository](https://github.com/productdevbook/port-killer)
